### Repaso Kicad

Comandos: 

(apuntes mío)

- **A:** herramienta agregar símbolo. 
- **ESC:** herramienta selección.
- **M:** mover componentes.
- **R:** rotar componentes.
- **X:** reflejar en el eje x.
- **Y:** reflejar en el eje y.
- **G:** mover componentes con todo lo que tiene conectado.
- **V:** asignar valor a componentes.
- **E:** editar/revisar hoja de de vida de componentes. 
- **F:** revisar huella asignada al componente.
- **CTRL + D:** duplicar componente. 
- **CTRL Z:** volver atrás. 
- **CMD + S/ CTRL + S:** guardar.
